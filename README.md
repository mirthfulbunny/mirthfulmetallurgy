# Mirthful Metallurgy

Steelmaking made interesting.

## Work-in-Progress Disclaimer

This mod is not ready to be used. While the license is permissive, I cannot recommend using it until it is ready.

There will be bugs, I am sure. There will be ugly placeholder textures. I know. You don't have to tell me.

## Features
*   **Generational Progression:** Explore the way that steel was made throughout history in a reasonably accurate (but not realistic) way.
*   **Grades of Steel:** Different grades of steel for different purposes, from tools to machinery, with varying properties and difficulty to manufacture.

Will also be looking into other metals as well in the future ;3c

## Requirements
*   Minecraft 1.20.1
*   Forge Loader

## Installation
1.  Download the mod jar from GitHub.
2.  Install Forge for the corresponding Minecraft version.
3.  Place the downloaded jar file into your `mods` folder.
4.  Launch Minecraft.

## Configuration
Configuration files can be found in `config/mirthfulmetallurgy.json`.
